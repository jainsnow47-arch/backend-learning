# Internet Basics

## What is Backend?

Backend is the part of an application that processes user requests, communicates with the database, applies business logic, and sends responses back to the client.

---

## What is a Server?

A server is a computer that is always connected to the internet. It receives requests from clients, processes them, and sends back responses.

---

## Client and Server

A client is a device or application that sends requests.

Examples:

- Web Browser
- Mobile App
- Desktop Application

A server receives those requests and returns the required data.

---

## Request and Response

The communication between a client and a server happens in two steps:

1. Client sends a Request.
2. Server processes the request and sends a Response.

Example:

Browser → "Give me Instagram Home Page."

Server → "Here is your homepage."

---

## Backend Architecture

```text
Browser (Client)
        │
        │ Request
        ▼
Backend Server
        │
        ▼
Database
        │
        │ Response
        ▼
Browser (Client)
```
